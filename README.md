<details>
<summary>🥡 How to Set Up</summary>

1. Clone the repo and install dependencies: `npm install`.
2. Copy the example environment file: `cp .env.example .env`.
3. Start the dev server: `npm run dev` and open the shown localhost URL.
4. Build for production when ready: `npm run build` (output in `dist/`).
5. Deploy the `dist` folder to your static host of choice.

</details>
